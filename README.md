# Arduino Simon Says

## Overview

This project is a Simon Says memory game built using an Arduino Mega, four LEDs, four push buttons, and a passive buzzer.

The Arduino generates a random sequence of colors and sounds. The player must repeat the sequence using the corresponding buttons.
## Features

* Randomly generated Simon Says sequence
* Four LED indicators
* Four player input buttons
* Audio feedback using a passive buzzer
* Success and failure sound effects
* Game over animation
* Start button functionality
* Increasing difficulty through longer sequences

## Components Used

* Arduino Mega 2560
* Breadboard
* 4 LEDs
* 4 Current-limiting resistors
* 4 Push buttons
* Passive buzzer
* Jumper wires

## How It Works

1. The game waits for the player to start.
2. A sequence of LEDs and tones is played.
3. The player repeats the sequence using the buttons.
4. Correct inputs advance to the next level.
5. Incorrect inputs trigger a game over sequence and reset the game.
